# Prompt Engineering

## Objective

Improve the quality of responses generated by the Student Support LLM through better prompt design.

## Original Prompt

```
You are a helpful student assistant.
Answer the student's question clearly.
```

### Test Question

```
What is Object-Oriented Programming?
```

### Example Response (Original Prompt)

```
Object-Oriented Programming (OOP) is a programming paradigm based on objects.
```

---

## Improved Prompt

```
You are an intelligent university student support assistant.

Instructions:
- Answer accurately and clearly.
- Use simple language suitable for university students.
- Explain concepts step by step.
- Provide examples where appropriate.
- If you are unsure, state that you are unsure instead of guessing.
- Keep responses concise but informative.
- Format answers using bullet points when helpful.
```

### Test Question

```
What is Object-Oriented Programming?
```

### Example Response (Improved Prompt)

```
Object-Oriented Programming (OOP) is a programming approach where software is organized around objects instead of functions.

The four core principles are:
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

Example:
A Car object can have attributes like color and speed, and methods such as start() and stop().
```

---

## Comparison

| Original Prompt  | Improved Prompt                 |
| ---------------- | ------------------------------- |
| Very short       | More detailed                   |
| No formatting    | Uses bullets                    |
| No example       | Includes an example             |
| Generic response | More informative and structured |

## Conclusion

The improved prompt produces clearer, more detailed, and more consistent responses, making the Student Support LLM more useful for students.
